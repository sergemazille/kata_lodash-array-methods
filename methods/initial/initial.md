# initial

Returns all the items of an array **but** the last one.

## Properties

Mutation: `false`

Difficulty: `Easy`

Return type: `unknown[]`

## Arguments

- `array: unknown[]`

## Example

```typescript
const array = ['banana', 'apple', 'peach', 'cherry'];
initial(array); // => ['banana', 'apple', 'peach']
```
