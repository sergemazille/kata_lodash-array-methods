# tail

Returns all the items of an array **but** the first one.

## Properties

Mutation: `false`
Difficulty: `Easy`
Return type: `unknown[]`

## Arguments

- `array: unknown[]`

## Example

```typescript
const array = ['banana', 'apple', 'peach', 'cherry'];
tail(array); // => ['apple', 'peach', 'cherry']
```
