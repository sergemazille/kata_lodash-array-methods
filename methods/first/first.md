# first

Returns the first item of an array.

## Properties

Mutation: `false`
Difficulty: `Easy`
Return type: `any`

## Arguments

- `array: unknown[]`

## Example

```typescript
const array = ['banana', 'apple', 'peach', 'tomato'];

first(array); // => 'banana'
```

## Notes

Returns `undefined` if the array is empty.
