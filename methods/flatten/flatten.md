# flatten

Flatten the first level of subarrays in an array.

## Properties

Mutation: `false`

Difficulty: `Medium`

Return type: `unknown[]`

## Arguments

- `array: unknown[]`

## Example

```typescript
const array = ['banana', ['apple', ['peach'], 'tomato']];

flatten(array); // => ['banana', 'apple', ['peach'], 'tomato']
```
