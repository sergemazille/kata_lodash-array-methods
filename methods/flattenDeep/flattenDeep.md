# flattenDeep

Fully flatten the subarrays in an array.

## Properties

Mutation: `false`

Difficulty: `Medium`

Return type: `unknown[]`

## Arguments

- `array: unknown[]`

## Example

```typescript
const array = ['banana', ['apple', ['peach'], 'tomato']];

flattenDeep(array); // => ['banana', 'apple', 'peach', 'tomato']
```
